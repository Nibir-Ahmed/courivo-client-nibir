# Courivo

A full-stack online learning platform where users can share, explore, enroll, and manage courses.

Live Site: [https://courivo.netlify.app](https://courivo.netlify.app)

## Features

- Email/Password and Google authentication with Firebase
- Browse and filter courses by category
- Instructors can add, update, and delete their own courses
- Students can enroll in courses and track enrolled courses
- Dark/Light theme toggle with responsive design
